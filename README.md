# Mini-game: Player VS Monster
Simple game using Angular 7, NodeJS, Bootstrap 4 e FlexBox.


## Getting Started

Mini Project using:
- Angular 7
- NodeJS (ExpressJS)
- Bootstrap 4
- FlexBox

### Prerequisites

- Node >= 8
- NPM


## Running

### FrontEnd
Install all dependecies
```
cd frontend && npm install
```

run and open
```
ng serve -o
```

### BackEnd
Install all dependecies
```
cd backend && npm install
```

run and open
```
npm start or npm run start
```
