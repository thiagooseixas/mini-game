# Mini-game: Player VS Monster
Simple game using Angular 7, NodeJS, Bootstrap 4 e FlexBox.


## Getting Started

Mini Project using:
- Angular 7
- NodeJS
- Bootstrap 4
- FlexBox

### Prerequisites

- Node >= 8
- NPM


### Running

Install all dependecies
```
npm install
```

run and open
```
ng serve -o
```
